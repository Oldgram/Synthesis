\subsection*{Syntax highlighting}

Using \verb|\begin{minted}{language} ... \end{minted}|. Example for java:

\begin{minted}[linenos]{java}
public static int cherche(String a, String b){
    int occ = -1;
    for (int i = 0; i <= a.length()-b.length(); i++){
        if (a.startsWith(b, i)){
            occ = i;
            break;
        }
    }
    return occ;
}
\end{minted}
